# 用过的笔记管理软件

我探索过很多笔记管理软件，从最开始用 Vim 的时候使用 Vimwiki，后面转向自托管的 leanote。后面编辑器切换到 Emacs 尝试过 org-wiki，
最终现在开始使用 org-roam。org-roam 基于 Emacs + Orgmode 的 [Roam Research](https://roamresearch.com/) 开源版本。
