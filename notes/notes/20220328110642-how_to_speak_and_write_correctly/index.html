How to Speak and Write Correctly

2022-03-28 :: [Gray King]
  • tags: Learning English
  • source: Joseph Devlin. How to Speak and Write Correctly, 2007. http://archive.org/details/how_to_speak_and_write_correctly_librivox.

I found this book in my Kindle on the subway to work this morning. And remembered that I downloaded it free from the Kindle store years ago. For some reasons, maybe my English was not enough to read it, I haven’t read it yet. After read a little, I think it’s prefect for me for now.

The reasons is:

  1. With Kindle, it provides some tooltips over the new words, so I can read it without a barrier.
  2. Also it provided me some new words to remember, as I don’t like to remember new words.
  3. I found some audios from archive.org, so I can listen to it how to pronounce the new words.

The steps I read this book should be:

  1. Read it on my Kindle, mark some words to remember.
  2. Listen to the audios, and read again.
  3. Take notes with my own words.
