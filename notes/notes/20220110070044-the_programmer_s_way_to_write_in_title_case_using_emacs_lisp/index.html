The Programmer’s Way to Write in Title Case Using Emacs Lisp

2022-01-10 :: [Gray King]
  • tags: Learning English,Title Case
  • source: “The Programmer’s Way to Write in Title Case Using Emacs Lisp.” Accessed January 10, 2022. https://hungyi.net/posts/programmers-way-to-title-case.

Genernal correct title cased phrase:

  1. Uppercase the first letter of most words

    e.g. “There Is No Spoon”

  2. Not capitalize ‘small’ and ‘unimportant’ words

    e.g. “Long Live the King”

  3. Always capitalize the first and the last words, even if they’re small

    e.g. “The Land and Save We Live On”

  4. Capitalize sub-phrase as if they were their own title

    e.g. “The Lonely Reindeer: A Christmas Story”
