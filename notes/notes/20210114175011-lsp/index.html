C++ LSP

2021-01-14 :: [Gray King]
  • tags: Emacs,LSP,C/C++,CMake

通过如下命令生成 clangd 识别的编译配置文件

mkdir build
cd build
cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=1 ..
mv compile_commands.json ../

然后重启 M-x lsp-restart-workspace RET 即可。
