LeetCode101: 9. Palindrome Number

2022-03-18 :: [Gray King]
  • tags: LeetCode101,Math

The key idea is:

  1. To use \(log10(10^n) = n\) to get how many digits in the number.
  2. Then we need iterate \(n + 1\) times to compare each side.
  3. The digit in the left is \(\frac{x}{10^{n-i}} \mod 10\).
  4. The digit in the right is \(\frac{x}{10^i} \mod 10\).
class Solution {
public:
	bool isPalindrome(int x) {
		if (x < 0) {
			return false;
		}
		// failed at here
		if (x < 10) {
			return true;
		}
		int n = log10(x);
		int ld = pow(10, n);  // left div
		int rd = 1;           // right div
		for (int i = 0; i < (n + 1) / 2; i++) {
			// left                   right
			if (x / ld % 10 != x / rd % 10) {
				return false;
			}
			ld /= 10;
			rd *= 10;
		}
		return true;
	}
};
