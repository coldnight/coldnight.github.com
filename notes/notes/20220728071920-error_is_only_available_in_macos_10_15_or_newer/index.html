error: is only available in macOS 10.15 or newer

2022-07-28 :: [Gray King]
  • tags: Flutter,GUI,macOS
  • souce: https://github.com/flutter/flutter/issues/73122

To solve this problem we should specify MACOSX_DEPLOYMENT_TARGET：

I would imagine that there is Apple documentation on managing build settings in Xcode, but I don’t have a link offhand. There’s no Flutter-specific documentation of the process, if that’s what you mean; it isn’t any different in a Flutter macOS application as it would be any other macOS application.

If you don’t want to use Xcode, you can change MACOSX_DEPLOYMENT_TARGET directly in Runner.xcodeproj/project.pbxproj.
