Solana 101: 2. Anchor

2022-11-22 :: [Gray King]
  • tags: Solana 101: Create an Escrow dApp

链上数据存储⌗

账号所有权⌗

  • 账号可以通过 Metadata 指定一个 Program 作为拥有者；
  • Program 作为拥有者可以：
    1. 修改账号数据；
    2. 使用该账号签名。

PDA(Program Derived Address) 1⌗

  1. PDA 只有一个类似公钥的地址，但是没有对应的私钥；
  2. 可以通过 getProgramAccounts 获取一个 Program 的 PDA。

使用 PDA 进行合约链上存储：

  1. 通过 SystemProgram.createAccount 进行创建2；
  2. 把指定 Program 作为新账号的 Owner；
  3. 将 PDA 传递给 Program 进行操作。

创建一个计数器⌗

  1. 部署智能合约；
  2. 创建一个 PDA 并进行初始化；
  3. 增加计数器；
  4. 其他账号获取 PDA，并增加计数器；
