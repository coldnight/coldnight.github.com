C++ 同步原子（atomic）

July 16, 2021

C++ 采用了顺序一致的原子变量，很像Java 同步原子（volatile）(与 C++ volatile 没有关系）。

atomic<int> done;

// Thread 1                    // Thread 2
atomic_store(&done, 1);        while(atomic_load(&done) == 0) { /* loop */ }
